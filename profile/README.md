# 영선북스

영선북스 도서의 예제 코드, 실습 자료, 보조 프로젝트를 관리하는 공식 GitHub 조직입니다.

## 소개

이 조직은 영선북스에서 출간하거나 준비 중인 IT 도서, 실습형 가이드, 101 시리즈의 예제 코드를 관리하기 위한 공간입니다.

각 저장소는 특정 도서나 학습 시리즈와 연결되어 있으며, 독자가 책을 보면서 직접 실행해볼 수 있는 코드와 실습 자료를 제공합니다.

## 저장소 구성

저장소는 보통 다음과 같은 목적을 가집니다.

- 도서 예제 코드
- 실습 문제 및 참고 해설
- 샘플 프로젝트
- 보조 자료 및 템플릿 코드

## 저장소 이름 규칙

저장소 이름은 영문 소문자와 하이픈을 사용합니다.

예시: `python-automation-101`, `fastapi-101`, `llm-api-101`, `rag-benchmark-101`

## 라이선스

각 저장소의 코드는 학습과 실습을 돕기 위한 목적으로 제공됩니다.
자세한 사용 조건은 각 저장소의 `LICENSE` 파일을 확인해 주세요.
